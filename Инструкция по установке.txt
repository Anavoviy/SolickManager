Приветсвую всех, кто решился скачать моё приложение для учёта работы компьютерной мастерской.
Здесь будет описано поэтапно, что необходимо сделать для того, чтобы корректно установить моё приложение на свой ПК или сервер.

1.Установите сервер PostgreSQL на ваше устойство, где будет храниться база данных
- необходимо перейти по ссылке https://www.enterprisedb.com/downloads/postgres-postgresql-downloads
- скачать самую последнюю версию для вашей ОС (я делал на 15.2)
- установите сервер на ваше устройство (при установке обязательно запомните, а лучше запишите логин и пароль от сервера)

2. Установите программу для работы с PostgreSQL (я рекомендую скачать DBVeaver - https://dbeaver.io/download/)

3. Запустите устанвленную программу и создайте подключение к вашему серверу

4. Запустите файл создания базы данных, который идёт вместе с приложением, он называется CreateDB.sql

5. Теперь скопируйте следующую строку и заполните её данными вашего сервера (без квадратных скобок):
   Host=[ip-адрес_вашего_сервера];Port=[порт, по умолчанию 5432];Database=SolickManager;Username=[имя для подключения, по умолчанию postgres];Password=[пароль от сервера, который вы задавали приустановке]

6. Откройте файл SolickManagerContext.cs и вставьте скопированную строку в следующее место:
	protected override void OnConfiguring(DbContextOptionsBuilder optionsBuilder)
        => optionsBuilder.UseNpgsql("[скопированная строка]");

7. Сохраните файл и запускайте приложение через ярлык  SolickManager
